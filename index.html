📖 Словарь

Главный экран приложения. Все слова, фильтры и быстрые действия.

Шапка:

  • + Добавить — открывает форму добавления слова
  • ⋯ Меню — сохранить проект, очистить словарь

Фильтры и сортировка:

  • Часть речи, статус (выученные / не выученные), уровень (A / B / C)
  • Сортировка по дате добавления, алфавиту, уровню, корню (שורש)
  • 📖 Текущий рассказ — показывает только слова, добавленные из открытого рассказа (по sourceStoryId). Становится доступным когда открыт рассказ.
  • 🔤 Фильтр по корню — клик на тег ש׳ в карточке или кнопка «Найти однокоренные» в паспорте фильтрует словарь по корню. Бейдж показывает количество слов с этим корнем. Кнопка ✕ сбрасывает фильтр.
  • 🔷 Фильтр по биньяну — клик на фиолетовый тег биньяна в карточке или кнопка «Найти в биньяне» в паспорте. Бейдж показывает количество глаголов этого биньяна. Работает параллельно с фильтром по корню.
  • Поиск глаголов — поиск по инфинитиву (לעזור) находит слово по корню (עזר). Также ищет по сохранённым формам слова.

Карточка слова:

  • Клик — открыть паспорт слова
  • Зелёная метка слева — слово отмечено как выученное
  • ש׳ корень — синий тег рядом с частью речи; клик фильтрует словарь по этому корню
  • 🔷 биньян — фиолетовый тег (только у глаголов); клик фильтрует по биньяну
  • Дата — дата добавления слова (мелко слева от кнопок); объясняет порядок при сортировке «По дате»
  • 🔊 — озвучить
  • 📝 — иконка означает что у слова есть примечание (наведи чтобы увидеть)
  • 🗑 — удалить (с возможностью отмены)

💡 Глаголы в словарной форме — прошедшее время, 3-е лицо, м.р. (הוא). Примеры сгруппированы по трём уровням: A, B, C. В карточке слова показывается текущий уровень (яркий бейдж) и максимальный доступный (приглушённый), если они отличаются.

🪪 Паспорт слова

Открывается кликом на слово в словаре, истории или флэшкарте. Содержит всю информацию о слове.

  • Иврит, никуд, транскрипция, перевод, часть речи
  • 🔗 Источник — если слово добавлено из рассказа или «Мой текст»: показывает исходную фразу (иврит + перевод) и кнопку перехода к тексту. Клик открывает текст и прокручивает к исходной фразе с синей подсветкой.
  • 📝 Примеры A / B / C — три уровня сложности, каждый хранится отдельно. Паспорт всегда открывается на табе A. Кнопка 🔄 рядом с табом — перегенерировать примеры этого уровня с учётом активного сценария (старые заменяются). Кнопка 🔄 Все уровни — перегенерировать все существующие уровни последовательно.
  • ⬆️ Сгенерировать уровень — AI создаёт примеры следующего отсутствующего уровня (A→B→C). Если активен сценарий — примеры создаются в его контексте.
  • Подсветка слова — в каждом примере целевое слово автоматически выделяется синим. Алгоритм: точное совпадение → подстрока (учитывает приставки ו/ב/ל) → корневые согласные (שורש).
  • Грамматика — для глаголов: словарная форма [инфинитив, наст., буд.] : биньян
  • 📝 Примечание — свободное поле для заметок. Форматирование сохраняется. Иконка 📝 в карточке словаря сигнализирует о наличии.
  • 🔤 שורש (корень) — корень в формате פ.ע.ל. Определяется автоматически. Кнопка «Найти однокоренные» фильтрует словарь по корню.
  • 🔷 בניין (биньян) — только для глаголов. Dropdown с 7 биньянами, AI определяет автоматически. Кнопка «Найти в биньяне» фильтрует словарь.
  • 🔍 Pealim — доступен для любого слова. Открывает русскую версию pealim.com/ru/search с поиском по слову — спряжения, все формы, словарные статьи.
  • 💬 Чанк — короткий речевой блок с этим словом. Добавляется вручную или через ✨ AI. Тренируется в режиме Чанки во флэшкартах.
  • ★ — добавить слово или фразу в ⭐ Избранное
  • 🔍 Pealim — открывает поиск по слову на pealim.com в новой вкладке (спряжения, формы, словарные статьи).

💡 Перегенерация примеров + смена сценария словаря (🎭) = один и тот же глагол в разных жизненных контекстах.

+ Добавить в шапке словаря. Два способа:

🤖 Через Claude (рекомендуется)
  1. Выберите язык ввода: 🇮🇱 иврит или 🇷🇺 русский
  2. Введите слово (опционально — добавьте контекст)
  3. «🤖 Заполнить через AI» → Claude заполнит все поля + 3 примера уровня A
  4. «✅ Добавить в словарь»
📦 Из текста (массово)

🎓 Обучение → 📋 Мой текст → вставьте текст → кликайте на незнакомые слова — добавляйте в словарь напрямую.

🎓 Режимы обучения

Все режимы доступны в меню 🎓 Обучение в порядке рекомендуемого использования:

📖 Истории

Claude сочиняет рассказ или диалог из ваших слов. Если активен сценарий — история создаётся в его контексте (об этом сообщает подсказка под заголовком). Два режима: автоматически из всех слов или выбрать вручную.

📋 Мой текст

Вставьте любой реальный текст на иврите. Claude разобьёт на фразы и переведёт. Оранжевые слова — незнакомые (добавить в словарь). Синие — уже в словаре (паспорт). Кликабельны все слова — словоформы автоматически привязываются к словарной форме, включая форму סמיכות. Два состояния раздела: когда нет открытого текста — показаны поле ввода и архив; когда текст открыт — они скрываются и весь экран занимает текст. Кнопка «← Архив» вверху возвращает к списку текстов. При переходе в другие разделы (словарь и т.д.) в навбаре появляется кнопка «← Вернуться к тексту» — мгновенно возвращает к открытому тексту на ту же позицию прокрутки.

  • 📄 Один текст — вставьте весь текст целиком; поле автоматически растягивается по высоте.
  • 📦 Собрать из частей — для длинных текстов (WhatsApp-переписка, SMS). Добавляйте части по очереди кнопкой «➕ Добавить часть», затем «📋 Обработать все части».
  • Смысловые блоки — если разделить текст пустыми строками, каждая группа строк между ними станет одним неделимым блоком. Переносы внутри блока сохраняются при отображении. Без пустых строк — авторазбиение по знакам препинания.
  • ✂️ Редактор структуры — открывается автоматически после обработки нового текста. Позволяет скорректировать разбиение на фразы до начала учёбы: 🔊 озвучить фразу, ✏️ редактировать текст (пустая строка внутри = разделить на две фразы), ↕ объединить со следующей фразой. Выделение фрагмента показывает всплывающий AI-перевод. Кнопка «Пропустить» — перейти в учебный режим без изменений. Если структура уже есть — редактор не показывается.
  • 🔍 Проверить текст — грамматическая проверка. Работает в обоих режимах: читает текст из поля или из добавленных частей.
  • Авто-очистка поля — после успешной обработки поле ввода очищается автоматически. Результат остаётся на экране; при переходе в режим обучения старый текст не мешает.

Управление видимостью текста:

  • ☑ Иврит / ☑ Русский — два независимых чекбокса в шапке задают начальное состояние для всех строк. По умолчанию: иврит виден, перевод скрыт. Настройка сохраняется между сессиями.
  • Клик на иврит-строку — показать / скрыть иврит только этой строки (не влияет на перевод).
  • Клик на перевод — показать / скрыть перевод только этой строки (не влияет на иврит).
  • Двойной клик на строку — открыть оба языка сразу.
  • 🔍 — кнопка в каждой строке, открывает эту фразу в Focus Reader.
  • ✏️ — редактировать иврит и/или русский перевод фразы. Если изменён только перевод — сохраняется сразу без AI. Если изменён иврит — перевод обновляется автоматически. После сохранения страница остаётся на той же фразе.
  • 🗑 Удалить текст — при удалении текста из архива автоматически удаляются все слова, добавленные только из него. Слова, связанные и с другими текстами, остаются (привязка убирается). Тост сообщает сколько слов удалено и отвязано. Кнопка «↩️ Восстановить» полностью откатывает удаление.

Архив текстов:

  • Поиск — фильтрация по названию в реальном времени.
  • Сортировка — четыре кнопки над строкой поиска: 🕒 Новые (по умолчанию), 🕒 Старые, А→Я, Я→А. Выбор сохраняется между сессиями. Работает вместе с поиском.

🃏 Флэшкарты

Классические карточки: слово → перевод + примеры. Настройки: направление, источник, статус, количество (5/10/20/Все). Примеры показываются для текущего уровня слова. Если слово из рассказа — исходная фраза показывается первой. Тулбар: 📖 Паспорт, ⛶ Zoom, 🔗 Контекст. Кнопки оценки ✓ Знал / ✗ Не знал накапливают статистику. Переключатель 📝 Слова / 💬 Чанки — меняет режим тренировки. В режиме Чанки тренируются только слова у которых заполнен чанк; счётчик показывает сколько таких слов в текущей выборке. Направление (ИВ→РУ / РУ→ИВ / Случайно) работает в обоих режимах. Живой счётчик над карточкой: прогресс, количество знал/не знал, % в цвете. Финальный экран: % результата, плитки, список незнакомых слов, кнопка «🔁 Повторить незнакомые».

🎙 Аудирование

Без API. Фраза озвучивается — оцените: ✓ Понял / ~ Частично / ✗ Не понял. Настройки: источник (Все / Избранное / Текущий рассказ), статус (Все / Невыученные / Выученные), количество, чекбокс «Только из текстов». Чекбокс комбинируется с любым источником. Подсказка работает для всех источников включая Избранное. Живой % обновляется после каждой оценки. Финальный экран: % в цвете, плитки, фразы для повторения под блюром, кнопка «🔁 Повторить ошибки».

⭐ Избранное

Фразы из разметки аудио (кнопка ⭐ у каждой размеченной фразы) сохраняются с точными таймкодами — воспроизводится ровно нужный фрагмент. Также можно добавлять из паспорта слова (➕) и историй. В карточке избранного: редактирование текста и перевода по клику, подстройка таймкодов ±0.1с, кнопка → к тексту — мгновенный переход к тексту-источнику с подсветкой нужной фразы. Доступны 3 режима тренировок: флэшкарты, аудирование, express — везде воспроизводится оригинальный аудиофрагмент.

📝 Тест

Выбор перевода из 4 вариантов. Настройки: направление, источник, количество (5/10/20/Все). Счётчик слов рядом с кнопкой «▶️ Начать». Результат: % правильных + список ошибок с паспортом.

🧠 SRS — интервальное повторение

Алгоритм SM-2: чем лучше помните — тем реже показывает. Бейдж показывает количество слов к повторению сегодня. Оценки: 😫 Забыл → 😕 Трудно → 🙂 Нормально → 😊 Легко.

⚡ ExpressTest

Быстрая самопроверка за 1–5 минут. Три режима: 📝 Слова / 💬 Фразы / 🔊 Аудио. Оценки: ✓ Знаю / ~ Не уверен / ✗ Не знаю.

⭐ Избранное

Отмечайте слова и фразы, которые требуют особого внимания.

Как добавить ⭐:

  • Паспорт слова — ★ у заголовка (слово целиком) или ★ у конкретной фразы
  • Истории — ★ у любой строки (фраза привязывается к слову)

Быстрый запуск тренировки из Избранного:

  • 🃏 Флэшкарты, ❓ Квиз, 🎙 Аудирование, ⚡ Express — кнопки в нижней части вкладки «⭐ Избранное». Используют только фразы из Избранного.

💡 Если у фразы в Избранном есть аудио-разметка — во всех режимах воспроизводится оригинальный фрагмент.

  • Уровень и формат — начальный / средний / продвинутый, рассказ или диалог
  • Стиль — произвольные пожелания («с юмором», «разговорный иврит»)

  • Оранжевые слова — новые от AI (клик → добавить в словарь)
  • Синие слова — уже в словаре (клик → паспорт слова)

Возможности:

Окно текста:

  • Шапка — название рассказа всегда видно сверху, не прокручивается
  • Тело — фразы прокручиваются внутри, само окно остаётся на месте

🔍 Focus Reader + Аудио

Открывается кнопкой 🔍 у каждой строки — показывает фразу крупно по центру экрана. Иврит и перевод управляются независимо: клик на иврит — показать/скрыть иврит; клик на перевод — показать/скрыть перевод. Начальная видимость берётся из глобальных чекбоксов «Иврит / Русский». При переходе между фразами и повторном прослушивании состояние видимости (показан/скрыт) сохраняется — не сбрасывается автоматически. При закрытии страница скроллит обратно к этой строке. Аудиоплеер скрывается на время работы Focus Reader и восстанавливается при закрытии.

Плеер FR визуально и функционально идентичен основному плееру текста — те же две строки управления, тот же набор кнопок. Кнопка 🎙 (красная круглая) в панели FR открывает встроенный диктофон: запись голоса, воспроизведение оригинала, сравнение. Разрешение на микрофон запрашивается один раз — стрим сохраняется между сессиями FR.

Встроенная панель управления аудио прямо в фокусе:

  • ⏮ ⏭ — предыдущая / следующая фраза (без воспроизведения)
  • 🔁 — повторить текущую фразу
  • ▶ — перейти к следующей и воспроизвести
  • ⏩ — непрерывное воспроизведение с текущей (повторно — пауза)
  • 🎯 — режим обучения: пауза после каждой фразы (shadowing). Фраза подсвечивается оранжевым во время паузы.
  • 📝 — режим разбора (Commentary). Появляется если загружен commentary-файл. Переключает режим без авто-старта: остановитесь на нужной фразе → нажмите 📝 → вручную запустите ▶ или 🔁 → прослушайте комментарий → нажмите 📝 снова → продолжайте основной текст с той же позиции. В активном состоянии кнопка выделена фиолетовым.

  • ▶ Оригинал — воспроизвести оригинал / TTS
  • ▶ Моя запись — воспроизвести свою запись

🎬 Сценарии

Для чего нужен сценарий: создать историю на основе конкретной жизненной ситуации и подобрать слова для неё. Сценарий не добавляет слова в словарь автоматически — словарь пополняется кликом по словам в Focus Reader и My Text.

Пайплайн:

  1. Опишите ситуацию на русском → нажмите 🎬 Создать
  2. AI генерирует: название, описание, роли, действия, эмоции, список слов
  3. Отредактируйте любые поля и список слов вручную (галочки, правка)
  4. Нажмите 💾 Сохранить сценарий — сценарий попадает в архив
  5. Нажмите 📖 Открыть в историях — переход в раздел «Истории»
  6. В разделе «Истории» выберите режим 🎬 Слова из сценария → выберите сценарий → нажмите ✨ Сгенерировать историю
  7. Отредактируйте текст в редакторе → нажмите 📋 Открыть в учебном режиме
  8. Дальше — полный My Text pipeline: разбивка на фразы, перевод, аудио, разметка, Focus Reader, Shadowing, обучение

Архив сценариев — находится в разделе «🎬 Сценарии» под формой создания. Каждый сохранённый сценарий можно:

  • 📂 Открыть — загрузить в редактор Step 2 для правки
  • 🗑 Удалить — удалить из архива

💡 Один сценарий — несколько историй: сохраните сценарий и генерируйте разные истории на его основе (рассказ, диалог, разные уровни сложности).

🎧 Оригинальное аудио

Рассказ становится полноценным учебным материалом: текст + живое аудио как единое целое.

  • Загрузите аудио в плеер «Мой текст» — медленную и/или нормальную запись.
  • 🎯 Разметка аудио — кнопка под текстом. Открывает модал с аудиоплеером и списком фраз. Подсказка по управлению — кнопка ? рядом с заголовком.
    • 🎯 Старт → ⏹ Стоп — кнопка работает по клику: первый клик фиксирует начало и запускает воспроизведение, кнопка становится красной «⏹ Стоп»; второй клик фиксирует конец и ставит паузу. Удобно для длинных фрагментов и commentary. Если фраза уже размечена — кнопка показывает «✏️ Перезахват».
    • Пробел (зажать → отпустить) — PTT-режим для коротких фраз: нажатие фиксирует начало, отпускание — конец.
    • ⚙️ Тонкая подстройка — кнопка у каждой размеченной фразы. Раскрывает панель сдвига границ: ±0.1с и ±0.5с для START и END независимо. Кнопка 🔊 рядом с каждой границей воспроизводит ~1.5с вокруг точки для точной проверки. Изменения сохраняются автоматически.
    • 📍 Перейти — переводит аудио на позицию начала фразы.
    • 🔊 Прослушать — воспроизводит размеченный отрезок целиком.
    • ✕ Сбросить — удаляет разметку одной фразы.
  • Независимые треки — 🐢 медленный, 🚶 нормальный и 📝 разбор размечаются отдельно. Переключение между треками не удаляет уже сделанную работу. Счётчик в кнопке «Разметка аудио» показывает количество размеченных фраз для каждого трека.
  • 🎵 Переключатель трека — появляется в навбаре когда загружен хотя бы один аудиофайл со slow/normal разметкой. Один файл — бейдж (🐢 или 🚶). Два файла — выпадающий переключатель. Выбор глобальный: применяется во всех режимах обучения.
  • 📝 Разбор — Commentary — третий независимый трек для пофразового разбора:
    • Загрузите аудиофайл разбора через вкладку «📝 Разбор» в плеере «Мой текст».
    • Разметьте через «Разметка аудио» — выберите трек «📝 Разбор» в select. Разметка commentary независима от slow/normal.
    • Кнопка 📝 в Focus Reader — появляется автоматически когда загружен commentary-файл. Переключает режим (без авто-старта). В активном состоянии выделена фиолетовым.
    • Сценарий: остановиться на нужной фразе → нажать 📝 → запустить ▶ → прослушать комментарий → нажать 📝 снова → продолжить основной текст с той же позиции.
  • 🐢🚶📝🎬 в архиве текстов и рассказов — интерактивные индикаторы у каждой записи:
    • 🐢✓ / 🐢— — медленный трек загружен. ✓ (зелёный) — разметка есть; — (серый, приглушённо) — файл есть, но не размечен. Клик: открывает и запускает трек.
    • 🚶✓ / 🚶— — нормальный трек. Аналогично.
    • 📝✓ / 📝— — трек разбора. Аналогично.
    • 🎬 — есть видео-разбор (только «Мой текст»). Клик: открывает и сразу запускает YouTube-плеер.
  • Оригинальное аудио во всех режимах — флэшкарты, аудирование, ExpressTest, паспорт слова, избранное: если для фразы есть размеченный оригинальный фрагмент — воспроизводится он, иначе TTS.
  • 🔁 A–B loop — повтор фрагмента: под прогресс-баром появляется панель с двумя точками:
    • A и B — кнопки установки точек по текущей позиции
    • −/+ рядом с каждой точкой — точная подстройка ±0.5с
    • ▶ A–B — запуск бесконечного повтора; 🔁 Стоп — остановка
    • Между повторами — автоматическая пауза 2 секунды
    • ✕ — сброс обеих точек
    • Клик по цветной полоске под слайдером: 1-й = A, 2-й = B, 3-й = сброс
  • 🎙 Прослушать (плеер рассказа) — кнопка под рассказом открывает компактный плеер внизу экрана (высота ~100px, не перекрывает текст). Плеер устроен в две строки:
    • Строка 1: лейбл «🎙 N / M» → ползунок фраз (быстрый переход без воспроизведения) → ☆ отметить текущую фразу как сложную → ☆ N счётчик сложных / фильтр → кнопка ✕.
    • Строка 2 — навигация: ⏮ ⏭ — предыдущая / следующая фраза без воспроизведения (← →).
    • Строка 2 — воспроизведение: 🔁 — повторить текущую (Пробел); ▶ — следующая + воспроизвести (N); ⏩ — непрерывно с текущей (Ctrl+Пробел, повторно — пауза).
    • 🎯 Режим обучения — пауза после каждой фразы при ⏩. Рядом появляется inline-слайдер коэффициента (×0.5–×2.0, пауза 0.5–5 сек). Фраза во время паузы подсвечивается жёлтым — идеально для shadowing.
    • ⭐ Отметка сложных фраз — кнопка ☆ / ⭐ (или клавиша H) отмечает текущую фразу как сложную. Отмеченные фразы выделены красной полосой слева. Кнопка ☆ N показывает количество отмеченных; нажмите её чтобы переключиться в режим «только сложные» — идеально для точечного shadowing проблемных фраз. Метки сохраняются в тексте/истории. При закрытии плеера фильтр сбрасывается.
    Текущая фраза подсвечена зелёной рамкой; остальные фразы приглушаются — взгляд сам падает на активную строку. Если аудио размечено — воспроизводится оригинал, иначе TTS.
  • ✏️ Редактирование фраз — кнопка ✏️ рядом с каждой фразой в тексте. Открывает модал с двумя полями: иврит и русский перевод. Изменения сохраняются сразу; если изменён только иврит — перевод обновляется автоматически через AI.
  • ▶ Разбор (видео) — кнопка под рассказом. Открывает модал с встроенным YouTube-плеером. Настройка:
    • Вставьте YouTube-ссылку в поле URL — поддерживаются youtu.be/…, youtube.com/watch?v=… и голый ID.
    • Начало (сек) — с какой секунды начать. Если поле пустое, таймкод t= из ссылки подставляется автоматически. Ручное значение имеет приоритет.
    • Конец (сек) — необязательно; если указан, плеер остановится в этой точке.
    • ▶ Применить — загружает видео без сохранения. 💾 Сохранить — записывает в рассказ. ✕ Удалить — удаляет видео-разбор.
    • Кнопка 🔗 Открыть на YouTube всегда видна поверх плеера — полезна если канал запретил встраивание.
